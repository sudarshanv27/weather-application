\# 🌤️ Weather Web Application



A beautiful Flask web application that provides real-time weather information for any city using the OpenWeatherMap API.



\## ✨ Features



\- Search weather for any city worldwide

\- Real-time temperature in Fahrenheit

\- Weather conditions (Sunny, Rainy, Cloudy, etc.)

\- Humidity percentage

\- Wind speed

\- Beautiful responsive design

\- Error handling for invalid cities
